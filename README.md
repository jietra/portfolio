### 🔒 Distributed Identity Application (Private)
A cross‑platform identity system built with **React‑Native**, **Node.js**, and **Go**.  
Core capabilities:
- Distributed ID (DID) protocol  
- Private blockchain backend  
- One‑time verifiable codes (QR or NFC)  
- Zero‑knowledge attribute proofs (e.g., “over 18” without revealing age)  
- Secure mobile‑side encryption for credential generation  
- Post‑Quantum Cryptographic (PQC) protocols  
- Use cases: event tickets, access control, verifiable attributes  

Focus: privacy‑preserving identity and cryptographic verification.
